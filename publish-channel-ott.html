Introduction

Once you have created your Cloud Playout channel, there are three steps you need to perform in order to make it part of your {{ site.product_short }} app. These are detailed in the sections that follow.

Get information for the {{ site.product_short }} Channel

Before you proceed you will want to capture information that you will need in setting up your {{ site.product_short }} channel and playlist.

  1. Go to the channel details page by clicking on the channel in the channel list.
  2. If you have not already done so, click Activate to activate the channel. See Using Channel Actions for more details.
  3. After the channel has activated, use the main navigation to go to the Media module.
  4. At the top of the video list, you should see a new live video that was created for your Cloud Playout channel:

    Note that it will be a Live video with no associated image.

  5. Click on the video name to open the details.
  6. Save both the video ID and the channel ID (the long alphanumeric string that follows cloud-playout: in the Tags).

Note that the channel ID is also available in the Channel Details in the Cloud Playout module:

Create a {{ site.product_short }} Channel

This step must currently be performed in {{ site.product_short }} Classic.

To create a Channel in {{ site.product }}, follow these steps:

Basic data

  1. Log in to {{ site.product_short }} Classic.
  2. In the top navigation, select Channels.
  3. Click the Add new channel button.
  4. In the Basic Data tab, define a Name for your channel.
  5. Enter the Cloud Playout channel ID:

    To find the Cloud Playout channel ID, see Get information for the {{ site.product_short }} Channel.

  6. In the Content Rating dropdown menu, select your channel's content rating based on the Content Rating System of the country where your channel will be streaming. In the example image, it is for the US TV-Y.
  7. In the Genres Details section, click the Genres dropdown menu to select a genre for your channel. (Example: Horror, Drama, etc..)
  8. If you donâ€™t find the correct genre for your channel, you can create additional ones by clicking the Create Genre button.
    9. The Sub-Genres dropdown menu is not functional at this time.

Textual Data

The Textual Data section allows you to add displayable text for your Channel.

  1. Select the Textual Data tab.
  2. In the Title field, define a title for your channel.
  3. In the Description field, define a description for your channel.
  4. In the Synopsis field, define a synopsis for your channel.
    5. You can add a Textual Data section for every different language you will use in your channel.

Streams

In the Streams section you will set up streaming information for your channel

  1. In the Streams section, select Brightcove as the Provider and click Add new stream:
  2. Add the Steams information:
    • Stream Video ID: enter the video ID for the channel's Live stream
    • Devices to Publish: click Select/Unselect All Devices and then remove any that you do not want to publish to
    • Stream Type: select Live
    • DRM Type : select the DRM type you are using or Open
    • Stream Format: select HLS

Rights & Scheduling

  1. In the Rights & Scheduling section, add the following information:
    • Status: select Published
    • Allow Downloads: choose whether to allow viewers to download content (in most cases, the answer will be no)
    • Rights Type: select Free or whatever is appropriate for your content
    • Start Time: the time to make the channel available (choose a date in the past to make it available immediately)
    • End Time: the time the channel will become unavailable (you can set it far in future if you want it to run indefinitely)
    • Max Streams: You don't need to enter anything - leave it at 0
    • Devices to Publish: click Select/Unselect All Devices and then remove any that you do not want to publish to
    • Permitted locations: enter World or the areas you allow viewing
    • Denied locations: leave blank or enter areas where viewing is not permitted

Create the channel

  1. After you have entered all the necessary settings, click Create New Channel

Add the channel to a playlist

Because {{ site.product_short }} pages use playlists for content, you need to add your Cloud Playout channel to a playlist. The playlist can contain other videos, but typically it will contain only the channel so that it comprises the page contents.

  1. Go to the Media module in {{ site.product_short }} Studio
  2. Click Add New Playlist
  3. In the Create playlist dialog, enter a name for the playlist and select the Manual type. You can also add a reference ID and description, and have it show up in the sidebar if you wish.
  4. Click Save to save the playlist
  5. Go to the list of videos and click the name of the video associated with your channel to open the video details.
  6. Click the Add to dropdown and select Add to Playlist:
  7. In the Add to Playlist dialog, select the playlist you created for the channel and click Add:

Add channel to a {{ site.product_short }} page

The last step in putting your Cloud Playout channel into your {{ site.product_short }} app is the simplest one: all you need to do is add your playlist containing the channel video to page. See Managing Playlists on Pages for details.
