Présentation

Vous pouvez utiliser les compartiments AWS S3 soit comme destination unique pour une chaîne Cloud Playout, soit en plus d'une adresse RTMP ou de Brightcove Live. La page Configuration vous permet de configurer la configuration de votre/vos compartiment (s) une seule fois au lieu de chaque fois que vous créez un canal.

Configuration des buckets

Pour configurer un ou plusieurs compartiments S3 à utiliser avec Cloud Playout, suivez les étapes ci-dessous.

  1. Dans Cloud Playout, cliquez sur Configuration dans le menu de gauche:
  2. Cliquez sur Ajouter nouveau.
  3. Entrez le nom du compartiment, la clé d'accès, la clé secrète et la région.
  4. Cliquez sur Ajouter.

Pour plus d'informations sur la façon de générer une clé d'accès et une clé secrète, consultez How do I create an AWS access key ?
