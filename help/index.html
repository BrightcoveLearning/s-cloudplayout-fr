Rechercher une ou plusieurs chaînes

Si vous avez de nombreuses chaînes, vous pouvez utiliser la zone de recherche pour trouver celle (s) que vous souhaitez.

  • Détails
Ajouter une nouvelle chaîne
  1. Cliquez pour ouvrir la boîte de dialogue Créer une chaîne.
  2. Entrez un nom (en anglais uniquement, sans espaces ; ces caractères spéciaux sont autorisés: _.:/= + -).
  3. Saisissez une description de la chaîne.
  4. Cliquez sur le champ Date/Heure de début pour sélectionner une date et une heure de début.
  5. Si vous basculez sur, vous pouvez également saisir une date/heure de fin facultative
  6. Si vous ne souhaitez pas importer les 608 sous-titres codés intégrés aux vidéos, désactivez cette option. Voir Importer des légendes
  7. Basculer activé pour activer SSAI - l'option Live Input sera également supprimée. Si vous activez cette option, vous pouvez également activer Importer les points de repère Mid-Roll pour les publicités mid-roll et modifier la durée des publicités mid-roll, si vous le souhaitez.
  8. Basculer pour inclure des événements en direct dans votre chaîne. Vous devrez également fournir une URL pour le contenu de la ardoise ou utiliser la liste par défaut pour remplir le temps pendant lequel la diffusion en direct n'est pas disponible.
  9. Sélectionnez Mobile & Web ou Smart TV pour la sortie.
  10. Sélectionnez la région de sortie la plus proche de la majorité de votre public.
  11. Sélectionnez Brightcove Live, puis Create New Brightcove Live Channel comme destination de sortie.
  12. Cliquez pour créer la chaîne.
  • Détails
Supprimer une chaîne
  1. Sélectionnez une ou plusieurs chaînes en cochant les cases.
  2. Cliquez .
  3. Cliquez dans le modal de confirmation pour supprimer définitivement le (s) canal (s).
