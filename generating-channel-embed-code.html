Génération du code intégré pour une chaîne Cloud Playout

Pour générer le code intégré d'une chaîne Cloud Playout, vous allez utiliser le module Live. Pour plus de détails sur l'utilisation du module Live, consultez Création et gestion d'événements en direct à l'aide du module Live.

Deux types de code intégré sont disponibles:

  • Standard - code d'intégration iframe. Recommandé dans la plupart des cas. Ce type place le lecteur à l'intérieur d'une balise <iframe>.
  • Avancé: code intégré à la page. Permet au lecteur d'exister directement dans la page HTML, et non dans un iframe. Cela offre aux développeurs l'avantage d'accéder facilement au lecteur et aux propriétés et événements associés.

Il est également possible d'obtenir l'URL du lecteur publié. Cela est utile pour visualiser l'événement dans un navigateur.

Pour générer le code intégré d'un événement en direct, procédez comme suit:

  1. Ouvrez le module Live.
  2. Localisez l'événement en direct sur lequel la chaîne sera diffusée en streaming. En règle générale, l'événement se trouve dans l'onglet Standby et porte le nom Channel, sauf si vous avez sélectionné un événement spécifique à diffuser lors de la création de la chaîne.
  3. Cliquez sur le nom de l'événement pour ouvrir la page Control Room.
  4. Cliquez sur Publier et intégrer > Configuration du lecteur dans le menu de navigation de gauche.
    Remarque: Si le canal est configuré pour activer les publicités côté serveur (SSAI), Brightcove Player version 6.18.3 et supérieure doit être utilisé. Seuls les lecteur s de la version 6.18.3 et supérieure apparaîtront dans la liste déroulante.
  5. Cliquez sur l'URL du lecteur pour prévisualiser l'événement dans un navigateur ou copiez le code intégré approprié à utiliser.

Si le programme a été créé avec l'option Insertion d'annonces définie sur Activé, les options de publication incluront une liste déroulante Sélectionner la configuration de la publicité dans laquelle l'annonce la configuration à utiliser peut être sélectionnée.
